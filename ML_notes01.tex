	\section{Vectors, Vector Spaces and Vector Subspaces}
	\subsection{Vectors}
	The primitive idea of a vector is that, it is a quantity with magnitude and direction. But the abstract idea of a vector is that, it can be thought of as an ordered n-tuple. The way we interpret that, can be based on different applications of that vector. Suppose, we have a 5-component vector like, \[
	u = \begin{pmatrix}
		u_1 \\
		u_2 \\
		u_3 \\
		u_4 \\
		u_5
	\end{pmatrix}
	\]
	where each of the five components represent a health parameter of a patient. E.g. blood pressure, pulse rate etc. This idea is strictly tied to the application of the vector. In order to define the vector mathematically, we need an abstract concept of \textbf{`Field'}, that will lead us to the \textbf{`Vectors'}.
